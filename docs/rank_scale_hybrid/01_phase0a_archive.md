## 2. 已固定事实

1. Trigger Audit Delivery 已完成，结论为 `FAIL`。
2. 当前合法 `PASS trigger = 0`。
3. Bootstrap Sign-Stability Preregistration 已完成，但 Phase 0B 已不再因该 prereg 继续推进。
4. 原始 `T5_resid_persistence_M4` 已在当前仓库内恢复可执行。
5. 2008 的 T5 失败主因已裁决为 `UPSTREAM_DATA_EDGE_CASE_CONFIRMED`，不是原始 T5 算子炸裂的干净证据。
6. Benchmark Delivery 已落盘；T5 在 2017/2018/2020 的 `std(z)` 约为 `2.01 / 1.98 / 2.38`，EGARCH-Normal 在 2020 的 `std(z)` 约为 `0.81`。
7. Benchmark 在 2018 的 directionality 为负，而 T5 在 2018 为正，因此 benchmark family 未对 T5 形成完整材料性支配。

## 3. 已终止路径

以下路径在当前证据链下终止：

1. 继续寻找或包装 trigger。
2. crisis override。
3. hard switch。
4. MoE 或专家网络。
5. 以 Phase 0B 名义继续 bootstrap 数值跑批。
6. 把 trigger audit 的失败解释为“待后续推进”。

## 4. 新旧研究线边界

rank-scale hybrid 不是 Phase 0B 的续修，也不是旧 crisis architecture 的低配版本。它不依赖 trigger、regime classifier、override、hard switch 或 MoE。

新假说的唯一动机来自 Phase 0A 已落盘的两个互补事实：

1. T5 在部分窗口保留可用的 ordinal / rank ordering。
2. EGARCH-Normal 在 2020 展示更强的 cardinal / scale calibration。

因此，新研究线的问题被限定为：能否构造低自由度 `sigma_hybrid(t)`，同时继承 T5 的排序信息与 EGARCH-Normal 的尺度信息。该问题必须重新走预注册、最小实验和裁决流程，不得继承 Phase 0B 的任何入场资格。
